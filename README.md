# Selvstudium

Turningen ligger fortsatt litt på is (skulder og kne krangler litt), men trening på slackline er et greit alternativ.
Programmeringen går imidlertid greit (hodet krangler ikke), og i katalogene her ligger eksempler på selvstudium med Python & KI.
